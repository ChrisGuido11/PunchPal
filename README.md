# PunchPal - AI Boxing Coach App

PunchPal is a beginner-friendly boxing app that provides AI-generated personalized workout plans. Built with React Native, Expo, and powered by Grok AI.

## Overview

PunchPal solves the problem of expensive personal coaching by using AI to generate personalized boxing workouts based on user experience level, workout history, and progress. The app makes professional boxing training accessible to everyone.

## Features

- **AI-Generated Workouts**: Personalized boxing workout plans generated by Grok AI based on your level and history
- **Beautiful Design**: Modern minimalist design with dark theme, red and gold accents, and smooth animations
- **Progress Tracking**: Automatically tracks completed workouts and adjusts future recommendations
- **Smart Timer**: 3-minute rounds with automatic combo changes every 60 seconds
- **Onboarding**: Simple experience level selection (Beginner, Intermediate, Advanced)

## App Structure

### Screens

1. **Splash Screen** (`src/screens/SplashScreen.tsx`)
   - Animated entry with PunchPal branding
   - Routes to Onboarding or Home based on completion status

2. **Onboarding Screen** (`src/screens/OnboardingScreen.tsx`)
   - Boxing level selection (Beginner, Intermediate, Advanced)
   - Sets user preferences in Zustand store

3. **Home Screen** (`src/screens/HomeScreen.tsx`)
   - Displays AI workout generation with animated loading state
   - Shows workout card with glassmorphic design
   - Celebration animations when workout is ready

4. **Timer Screen** (`src/screens/TimerScreen.tsx`)
   - 3-minute round timer
   - Punch combo display that changes every 60 seconds
   - Round tracking and workout completion

### Components

- **WorkoutCard** (`src/components/WorkoutCard.tsx`)
  - Glassmorphic card with gradient glow effect
  - Displays workout details (duration, rounds, combos, difficulty)
  - Animated entrance with celebration micro-animations

### State Management

- **User Store** (`src/state/userStore.ts`)
  - Manages user profile and boxing level
  - Stores current workout and workout history
  - Persisted with AsyncStorage

### API Integration

- **Workout Generator** (`src/api/workout-generator.ts`)
  - Generates personalized workouts using Grok 3 Fast AI
  - Takes boxing level and workout history as input
  - Returns structured workout with combos and descriptions

## Tech Stack

- **React Native**: 0.76.7
- **Expo**: SDK 53
- **Navigation**: React Navigation (Native Stack)
- **State Management**: Zustand with AsyncStorage persistence
- **Styling**: NativeWind (TailwindCSS for React Native)
- **Animations**: react-native-reanimated v3
- **AI**: Grok 3 Fast (via OpenAI-compatible API)
- **Package Manager**: Bun

## Key Design Decisions

1. **Dark Theme**: Provides energy and focus for workout sessions
2. **Red & Gold**: Boxing-inspired color palette that conveys power and achievement
3. **Glassmorphic Cards**: Modern, premium feel with depth and hierarchy
4. **Smooth Animations**: All interactions use react-native-reanimated for 60fps performance
5. **Haptic Feedback**: Physical feedback on all interactions for better UX
6. **Safe Area Handling**: Proper insets for all device types

## Boxing Notation

Standard boxing punch notation used in the app:
- 1 = Jab
- 2 = Cross
- 3 = Lead Hook
- 4 = Rear Hook
- 5 = Lead Uppercut
- 6 = Rear Uppercut
